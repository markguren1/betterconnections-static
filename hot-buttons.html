Hot Buttons (Triggers)

Common triggers for each personality style and how to de-escalate quickly.

Driver

Triggered by: slow pace, indecision, lack of ownership.

Try this: be direct, offer 2 options, define next steps.

Connector

Triggered by: cold tone, isolation, being ignored.

Try this: acknowledge feelings, engage personally, listen first.

Analyst

Triggered by: unclear expectations, missing details, sudden changes.

Try this: explain the “why,” share data, allow processing time.

Harmonizer

Triggered by: conflict, public criticism, unpredictability.

Try this: reassure, lower emotional temperature, move gently.
